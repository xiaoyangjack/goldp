## 技术指标说明

### ATR (Average True Range)
- 周期：14天
- 用途：衡量市场波动性，用于动态调整网格间距

### MA (Moving Average)
- MA10：10日简单移动平均
- MA30：30日简单移动平均
- 用途：判断市场趋势方向

### 网格策略
- 上轨：价格 + 1.5倍ATR
- 下轨：价格 - 1.5倍ATR
- 网格数：6层

### 风控四层
1. 单笔交易最大亏损 ≤ 1%
2. 单日最大亏损 ≤ 3%
3. 单周最大亏损 ≤ 8%
4. 连续4笔亏损自动暂停
