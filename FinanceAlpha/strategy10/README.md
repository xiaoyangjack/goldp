# 策略10：另类数据驱动的事件驱动量化策略

## 策略简介

策略10是一个基于另类数据的事件驱动量化策略，通过AI技术解析转化为量化Alpha信号，构建事件驱动策略。系统支持多种另类数据源，包括供应链数据、电商与消费数据、卫星遥感数据、资金流数据、高管与股东行为数据等。

## 核心功能

1. **多源数据集成**：支持多种另类数据源的自动更新、清洗和因子化处理
2. **AI驱动策略**：使用机器学习模型分析数据，生成量化信号
3. **多种策略**：实现了景气度提前预判策略、短线事件驱动策略和行业轮动策略
4. **回测评估**：内置回测模块，评估策略性能
5. **自动化报告**：生成详细的策略报告

## 策略模块

### 1. 景气度提前预判策略

- **原理**：基于供应链数据、电商数据和资金流数据，使用机器学习模型预测股票价格走势
- **信号生成**：当模型预测上涨概率超过阈值时生成买入信号，反之生成卖出信号

### 2. 短线事件驱动策略

- **原理**：基于高管与股东行为数据，分析内幕交易活动
- **信号生成**：当净买入量和买入频率超过卖出时生成买入信号，反之生成卖出信号

### 3. 行业轮动策略

- **原理**：基于电商数据，计算各行业的景气度得分
- **信号生成**：选择得分最高的行业进行投资

## 数据接口

系统支持以下另类数据源：

- **供应链数据**：供应商交货时间、库存水平、生产率等
- **电商与消费数据**：销售额、销售量、客户满意度等
- **卫星遥感数据**：活动水平、存储容量等
- **资金流数据**：资金流入、流出、净流量等
- **高管与股东行为数据**：内幕交易类型、金额、交易人数等

## 技术实现

- **数据处理**：自动清洗、填充缺失值、标准化处理
- **特征工程**：添加滞后特征、滚动特征、动量特征
- **机器学习**：使用随机森林分类器预测价格走势
- **回测系统**：计算累计收益率、年化收益率、波动率、夏普比率、最大回撤等指标

## 配置说明

策略支持以下配置：

- **数据更新频率**：可配置为每日、每周、每月
- **策略参数**：如景气度预判的阈值、事件驱动的持有期等
- **回测参数**：初始资金、交易成本、基准指数等
- **风险参数**：最大仓位、最大回撤等
