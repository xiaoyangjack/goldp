## 界面应该显示的位置

### macOS
- 窗口应该显示在**屏幕顶部**
- 位置坐标 (0, 25) 表示：
  - **X = 0**: 屏幕最左侧

### 查找窗口的方法

1. **查看 Dock 栏**
   - 检查 Dock 栏中是否有 "GoldQuant" 或 "Python" 图标
   - 点击图标应该激活窗口

2. **使用 Mission Control**
   - 在触控板上三指向上滑动（或按 F3）
   - 查看所有打开的窗口
   - 寻找标题为 "📊 GoldQuant - 黄金量化研究系统" 的窗口

3. **使用 Command+Tab**
   - 按住 Command 键并多次按 Tab
   - 切换到 GoldQuant 应用

## 如果窗口仍然不可见

可能的原因：

1. **屏幕分辨率问题**
   - 窗口可能超出屏幕边界
   - 尝试重新启动程序，窗口应该会居中显示

2. **多显示器设置**
   - 如果使用外接显示器，窗口可能在主显示器上
   - 检查所有显示器的角落

3. **系统设置**
   - 确保在"系统偏好设置" > "调度中心"中
   - 勾选"显示器具有单独的空格"

## 界面功能

### 1. Header (顶部控制栏)

**左侧:**
- 📊 **GoldQuant** - Logo

**中间:**
- 📈 **数据源选择** - 下拉菜单选择交易品种
- 📅 **日期范围** - 设置回测的时间范围

**右侧:**
- ⚙️ **设置** - 打开设置对话框
- 🔄 **刷新** - 重新加载数据
- ▶️ **运行回测** - 开始回测分析

### 2. Content (图表展示区)

**图表标签页:**
- 📈 **净值曲线** - 显示策略净值变化
- 📊 **年度收益** - 显示各年度收益率对比
- 📉 **回撤曲线** - 显示策略回撤情况
- 🎯 **IC分析** - 显示因子预测能力
- 🔥 **参数热图** - 显示参数敏感性
- 🔗 **相关性** - 显示策略相关性

**注意:** 当前使用简化模式，图表显示为数据摘要。安装 PyQt6-WebEngine 可获得交互式图表。

### 3. Footer (底部状态栏)

**左侧:**
- ⚡ **状态** - 显示当前状态（就绪/加载中/回测中）
- 📁 **数据** - 显示当前数据源
- 📊 **条数** - 显示数据记录数
- 💰 **价格** - 显示价格范围

**右侧:**
- 进度条 - 显示回测进度

## 设置对话框

点击 **⚙️ 设置** 按钮打开设置对话框：

### 选项卡1: 📋 策略
- 勾选要使用的策略
- SMA均线、RSI回归、MACD动量、布林带、多因子

### 选项卡2: 📊 参数
- SMA参数（快线/慢线周期）
- RSI参数（周期/超买超卖阈值）
- MACD参数（快线/慢线/信号周期）
- 布林带参数（周期/标准差）
- 多因子参数（触发阈值）

### 选项卡3: 🛡️ 风控
- ATR止损倍数
- 目标波动率
- 波动率仓位管理
- DXY过滤

### 选项卡4: 📁 数据源
- 交易品种选择
- 日期范围设置

## 运行回测

1. **配置参数** (可选)
   - 点击 ⚙️ 设置，配置策略和参数
   
2. **开始回测**
   - 点击 ▶️ 运行回测
   - 进度条会显示回测进度
   
3. **查看结果**
   - 图表标签页会显示回测结果
   - 切换不同图表查看详细数据

## 故障排除

### 窗口不显示
- 检查 Dock 栏
- 使用 Mission Control (F3)
- 使用 Command + Tab 切换

### 图表显示为文本
- 这是正常现象
- 当前使用简化模式
- 如需交互式图表，安装 PyQt6-WebEngine

### 数据加载失败
- 检查网络连接
- 查看终端错误信息
- 可能是缓存数据已过期
